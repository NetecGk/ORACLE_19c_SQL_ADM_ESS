 # **Capítulo 2.** Gestion de Objetos de la Base de Datos
 
 ## [**Práctica 2.1 Usando DML para gestionar datos y transacciones**](Practica2_1.MD)
 

En esta práctica construirás una tabla temporal basada en el esquema HR para **insertar, actualizar y eliminar** registros. Aprenderás a **confirmar** (COMMIT) y **revertir** (ROLLBACK) cambios, así como a establecer **puntos de guardado** (SAVEPOINT) para controlar el flujo de una transacción. El objetivo es que domines el ciclo completo de trabajo con **DML** en Oracle (INSERT/UPDATE/DELETE + control transaccional), empleando **variables de sustitución** para crear scripts reutilizables.

 ## [**Práctica 2.2 Usando DDL para definir y modificar estructuras de tablas**](Practica2_2.MD)

En esta práctica explorarás los principales comandos **DDL (Data Definition Language)** de Oracle, que permiten **crear, modificar y eliminar objetos** en la base de datos.
Aprenderás a construir tablas desde cero, duplicar estructuras existentes, agregar y eliminar columnas, establecer restricciones básicas y eliminar objetos de forma segura. También practicarás cómo cambiar el estado de una tabla a **solo lectura (READ ONLY)** para evitar modificaciones temporales.
