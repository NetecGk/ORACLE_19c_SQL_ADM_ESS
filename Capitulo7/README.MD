
# **Capítulo 7.** Movimiento de Datos

## [**Práctica 7.1 Uso de Oracle Data Pump**](Practica7_1.MD)

En esta práctica aprenderás a **utilizar Oracle Data Pump** para exportar e importar objetos de base de datos. Realizarás una exportación completa del esquema **HR**, analizarás los archivos generados (.dmp y .log), y posteriormente realizarás la importación de la tabla `EMPLOYEES` hacia otro esquema (`userdp`). Además, reforzarás la comprensión de los **directorios lógicos de Oracle**, los **privilegios de acceso a Data Pump** y las **opciones de remapeo de esquemas**.

<br/><br/>


## [**Práctica 7.2 Carga de Datos con SQL*Loader**](Practica7_2.MD)


En esta práctica aprenderás a **utilizar SQL*Loader** en su modalidad **express y controlada mediante archivo `.ctl`**, para cargar datos desde archivos de texto hacia tablas de Oracle. Ejecutarás scripts de preparación, revisarás archivos de datos (`.dat`), cargarás información en tablas existentes y analizarás los mensajes generados por el proceso, comprendiendo así cómo Oracle maneja la carga masiva de datos desde fuentes externas.
